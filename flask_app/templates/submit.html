How Accurate are Stock Analyst Recommendations?

This application uses the yfinance API to retrieve data on historical prices as well as analyst buy/hold/sell recommendations.
These analyst recommendations are evaluated based on the distribution of returns after certain recommendation types have been made.

The API is capable of retrieving stock recommendations for most publicly traded, large-cap companies in the US.
However, some companies do not have analyst recommendation data available.

Some great companies to try are Nvidia(NVDA), Tesla(TSLA), AT&T(T), Apple(AAPL)
